## Odoo ERP คืออะไร

คำตอบที่สั้นกระชับ แต่อาจจะเหมือนกับคำตอบว่า ERP ทั่วๆไปก็คือ "ระบบซอฟต์แวร์แบบโอเพ่นซอฟต์ที่ประกอบด้วยโมดูลการใช้งานทางธุรกิจที่หลากหลายแต่เชื่อมโยงการทำงานกันเป็นระบบเดียว ซึ่งจะช่วยให้คุณสามารถจัดการกับธุรกิจของคุณได้อย่างมีประสิทธิภาพ"

ประกอบไปด้วยระบบงานต่างๆที่จำเป็นในการดำเนินงานทางธุรกิจเช่น ระบบลูกค้าสัมพันธ์ (CRM) ระบบซื้อ (Sales) ระบบขาย (Purchase) ระบบคลังสินค้า (Inventory) ระบบบัญชีการเงิน (Accounting) ระบบผลิต (Manufacturing) และ eCommerce เป็นต้น

ถ้าเทียบกับซอฟต์แวร์ในกลุ่มเดียวกันก็เช่น SAP, Microsoft Dynamic 365, Oracle Netsuite และอื่นๆ

แต่...

Odoo ไม่เหมือนกับระบบ Propritery ERP ทั่วไป ในบทความนี้เราจะมาเจาะจุดแข็งของ Odoo เมื่อเทียบกับระบบอื่นๆกัน 4 ข้อ

### 1. 100% Open Source แบบไม่จำกัดจำนวนผู้ใช้งาน

Odoo เป็นระบบ Open Source ที่ไม่มีค่าใช้จ่ายด้าน License (เมื่อใช้ Odoo CE + OCA + Ecosoft) ในขณะที่ ERP แบรนด์อื่นจะมีข้อจำกัดด้านจำนวนผู้ใช้งานและเนื่องจากราคา License ที่ค่อนข้างสูง หลายครั้งหน่วยงานจำเป็นต้องแชร์ผู้ใช้งานร่วมกัน ส่งผลต่อประสิทธิ์ภาพการทำงานขององค์กร

![Open Source](~/assets/images/post/odoo-erp-0.jpg)

Odoo เป้น Open Source Software และไม่จำกัดจำนวนผู้ใช้งาน ทำให้ใช้งานได้อย่างเต็มที่ เพิ่มความปลอดภัย และยังลดค่าใช้จ่ายในระยะยาว

### 2. ใช้สถาปัตยการรมซอฟต์แวร์แบบโมดูลาร์ที่ทันสมัย ยืดหยุ่นและปรับแต่งได้ง่ายกว่า

ตรงข้ามกับ ERP ส่วนใหญ่ที่เป้นแบบเนื้อเดียว Monolith ซึ่งปรับแต่งได้ยากกว่า และมักมาพร้อมกับฟังก์ชั่นงานทั้งหมด บ่อยครั้งเกินความจำเป็นและสร้างความสับสนแก่ผู้ใช้งาน

![Modular Architecture](~/assets/images/post/odoo-erp-1.png)

Odoo มีโครงสร้างระบบแบบโมดูลาร์ Modular (Architecture) แต่ละฟังก์ชั่นงานแยกเป็นโมดูลย่อยๆแต่เชื่อมโยงกัน ทำให้สามารถเลือกใช้และประกอบฟังก์ชั่นงานได้ตามต้องการ มีความยืดหยุ่นในปรับเติมเสริมแต่งในอนาคต สอดคล้องกับธุรกิจที่ปรับเปลี่ยนได้ตลอดเวลา

### 3. ชุมชนผู้ใช้งานขนาดใหญ่ รวมถึง App Store ขนาดใหญ่

ด้วยความที่เป็น Open Source + Modular Architecture ทำให้ชุมชนผู้ใช้งานของ Odoo ขยายตัวอย่างต่อเนื่อง และมีการร่วมกันสร้างสรรค์โมดูลต่างๆออกมาเป็นจำนวนมาก จนปัจจุบันเป็นถือเป็นชุมชนด้าน Business Application ที่มีจำนวนโมดูลให้เลือกใช้งานที่สุดในโลก

![Odoo Community](~/assets/images/post/odoo-erp-2.png)

และด้วยชุมชนขนาดใหญ่นี้เอง ทำให้มีบทความความรู้ต่างๆถูกเขียนและแชร์สู่สาธรณะเป็นจำนวนมาก ให้ผู้ใช้งานระบบ Odoo เลือกต่อยอดความรู้ได้แบบฟรีๆ

![Odoo App Store](~/assets/images/post/odoo-erp-3.png)

Odoo จึงเป็นตัวเลือกที่เป็นมิตรกับทั้งผู้ใช้งานและเจ้าของกิจการ และไร้ข้อจำกัดในการปรับตัวสู่อนาคต


### 4. Odoo Community Association (OCA)

แน่นอนว่าการเป็นระบบเปิด ทำให้มีโมดูลเกิดขึ้นเป็นจำนวนมาก แต่ปริมาณไม่ได้แปรผันตรงกับการมีคุณภาพ 

Odoo จึงมีการรวมตัวกันของนักพัฒนาที่มีคุณภาพก่อตั้งสมาคม Odoo Community Association (OCA) เพื่อกำหนดแนวทางการทำงานแบบมืออาชีพ ทำให้โมดูลเสริมต่างๆที่เกิดขึ้นภายใน OCA ได้รับการตรวจสอบคุณภาพ และสามารถติดตั้งกับ Odoo Core ได้โดยไม่มีปัญหา

![OCA](~/assets/images/post/odoo-erp-4.png)

การคงอยู่ของ OCA เป็นอีกความโดดเด่นสำคัญที่ทำให้ Odoo แตกต่างแม้แต่กับค่ายโอเพ่นซอร์ส ERP อื่นๆ และ Ecosoft เองเราก็เป็นท้้งสมาชิก ผู้ดูแลโมดูลบัญชีไทย และสปอนเซอร์ของ OCA เพราะเราเห็นความสำคัญของซอฟต์แวร์ที่มีคุณภาพและการแบ่งปัน


เพื่อให้เข้าใจ OCA ว่ามีความสำคัญอย่างไรกับการวางระบบ Odoo ERP ผมขอแนะนำให้อ่าน 2 บทความนี้
